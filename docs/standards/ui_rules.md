# トレーナーシステム UI/UX設計標準規約

## 1. 画面IDの命名規則
画面IDは、必ず「初代ポケモンの図鑑番号3桁（001〜151）」と「そのポケモンの英語タイプ名」を組み合わせたフォーマットにしなければならない。
例：ヒトカゲをモチーフにした画面なら SCR-004-FIRE とすること。

## 2. エラーメッセージの規定
入力エラー時のメッセージには、必ずポケモンバトルのテキスト構文を含めること。
例：「効果はいまひとつのようだ…」「急所に当たった！」など。無機質な「エラーが発生しました」等の表現は禁止する。

## 3. セキュリティ入力欄
パスワード入力欄で文字を隠す際のマスク文字は、アスタリスク(*)ではなく、必ず「モンスターボールの絵文字(🔴)」を使用すること。
